# Rhyto-BMI: Your Personal BMI Calculator

Rhyto-BMI is a simple and intuitive Body Mass Index (BMI) calculator that helps you track your health. With a clean and user-friendly interface, you can quickly calculate your BMI using either imperial or metric units.

## Features

*   **Imperial and Metric Units:** Switch between units with a single click.
*   **Real-time BMI Calculation:** Get instant feedback on your BMI.
*   **BMI Classification:** Understand your BMI with clear and concise classifications.
*   **Responsive Design:** Use it on any device, from your desktop to your smartphone.
*   **Informative Disclaimer:** Learn about the limitations of BMI and its use as a health indicator.

## Getting Started

1.  Clone the repository: `git clone https://github.com/Rhyto-BMI/Rhyto-BMI.git`
2.  Open `index.html` in your browser to run the application locally.
